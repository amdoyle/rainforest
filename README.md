== Rainforest

=== About: 
* Basic clone of Amazon
* Project allows users to create products, add descriptions, and set prices
* Includes sessions to log in to app

== Versions: 
* Ruby 2.3.0
* Rails 4.2.5

== Gems:
* Bcrypt - Used for user login

== Set-up: 
* Bundle install
* Run migrations

== Additional: 
* Currently, does not include any styling or tests
